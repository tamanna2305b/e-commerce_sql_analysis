# e-commerce_sql_analysis
# E-Commerce Sales SQL Business Analysis

## Overview

This project analyses an e-commerce sales dataset using SQL to answer business questions related to sales performance, customer behaviour, profitability and operational efficiency.

The project demonstrates SQL skills including filtering, aggregation, grouping, ordering, conditional logic and window functions.



## Business Questions

- Total revenue
- Average order value
- Top customers
- Revenue by country
- Revenue by city
- Revenue by product category
- Profit analysis
- Monthly sales trends
- Customer rankings
- Return rate analysis



## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- COUNT
- SUM
- AVG
- CASE WHEN
- LIMIT
- RANK()
- Window Functions



## Tools

- SQL (SQLite)
- Python
- Pandas
- Google Colab


## Key Insights

- Identified the highest-performing customer segments.
- Analysed revenue across countries, cities and product categories.
- Ranked customers based on total spending.
- Evaluated profitability and return rates.
- Generated recommendations to support business decision-making.
